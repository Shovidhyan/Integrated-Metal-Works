# 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/Shovidhyan/Integrated-Metal-Works.git
```

Go to project directory:

```bash
cd Integrated-Metal-Works
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

---

# 🏗️ Build Project

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

---

# 🌍 Deploy to GitHub Pages

Deploy command:

```bash
npm run deploy
```

---

# ⚙️ GitHub Pages Configuration

Vite configuration:

```ts
base: "/Integrated-Metal-Works/"
```

Router configuration uses:

```tsx
<HashRouter>
```
